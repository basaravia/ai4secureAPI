# Generalidades
Para inicializar el entorno virtual ejecutar

para crear el vnenv
`virtualenv nombre_de_tu_entorno -p python3`
despues para abir el venv `source nombre_de_tu_entorno/bin/activate` en el bash. Para salir 
`deactivate`

# Puertos
